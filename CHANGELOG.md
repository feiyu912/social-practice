# 更新日志 (CHANGELOG)

本文件记录项目的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.1] - 2025-12-13

### 修复
- 修复数据库 `user` 表缺少 `email` 和 `phone` 字段导致注册失败的问题
- 修复 `teacher/add.jsp` 中 HTML `<div>` 标签未闭合导致页面异常的问题
- 修复 `student/add.jsp` 表单显示学号字段名不一致的问题（`studentId` → `studentNumber`）

### 更新
- `db_schema.sql` 添加 `email` 和 `phone` 字段到 `user` 表
- 优化表单字段映射一致性

## [1.0.0] - 2025-12-13

### 新增
- 用户管理模块
  - 用户注册、登录、登出功能
  - 三种角色支持：管理员、教师、学生
  - 用户状态管理（启用/禁用）

- 活动管理模块
  - 活动发布、编辑、删除功能
  - 活动状态管理（招募中/进行中/已结束）
  - 活动名额和报名人数统计

- 报名管理模块
  - 学生活动报名功能（仅招募中活动可报名）
  - 教师审核功能（通过/拒绝）
  - 学生退选功能（仅待审核状态可退选）

- 小组管理模块
  - 创建小组功能（仅审核通过的学生可操作）
  - 加入小组功能
  - 退出小组功能
  - 解散小组功能（仅组长可操作）

- 日常任务模块
  - 学生任务提交功能
  - 教师查看学生任务功能

- 实践报告模块
  - 学生报告提交功能
  - 教师审核和反馈功能

- 成绩评定模块
  - 教师评分功能
  - 学生成绩查询功能

- 公告管理模块
  - 公告发布功能
  - 公告列表查看功能

- 系统管理模块（管理员）
  - 用户管理
  - 数据导入导出

### 技术特性
- 基于 SSM (Spring + Spring MVC + MyBatis) 架构
- 内嵌 Tomcat 服务器，无需额外部署
- 支持 MySQL 8.0 数据库
- 使用 Druid 数据库连接池
- JSP + JSTL 视图层
- Jackson JSON 数据处理
