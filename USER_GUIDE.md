## 系统概述

学生社会实践管理系统为高校社会实践活动提供全流程信息化管理，包括三种用户角色：

- **管理员**: 系统管理、用户管理、活动管理、公告管理
- **教师**: 活动发布、报名审核、报告审核、成绩评定
- **学生**: 活动报名、小组管理、任务提交、报告提交、成绩查看

### 系统访问

- 访问地址: http://localhost:8080
- 登录页面: http://localhost:8080/login
- 注册页面: http://localhost:8080/register

## 管理员使用指南

### 1. 登录系统

使用管理员账号登录：
- 用户名: `admin`
- 密码: `123456`

### 2. 管理员首页

登录后进入管理员首页，可以看到以下菜单：

```
├── 首页
├── 学生管理
├── 教师管理
├── 活动管理
├── 公告管理
├── 系统日志
└── 数据导入导出
```

### 3. 学生管理

#### 3.1 查看学生列表

点击"学生管理"进入学生列表页面，显示所有学生信息：
- 学号
- 姓名
- 性别
- 班级
- 联系方式
- 邮箱

#### 3.2 添加学生

1. 点击"添加学生"按钮
2. 填写学生信息表单：
   - 用户名（登录账号）
   - 密码
   - 真实姓名
   - 学号
   - 性别
   - 班级
   - 联系电话
   - 电子邮箱
3. 点击"提交"保存

#### 3.3 编辑学生

1. 在学生列表中找到目标学生
2. 点击"编辑"按钮
3. 修改学生信息
4. 点击"保存"更新

#### 3.4 删除学生

1. 在学生列表中找到目标学生
2. 点击"删除"按钮
3. 确认删除操作

### 4. 教师管理

操作流程与学生管理类似，主要管理教师的以下信息：
- 教师工号
- 真实姓名
- 所属院系
- 联系电话
- 电子邮箱

### 5. 活动管理

管理员可以查看、编辑、删除所有活动（包括教师创建的活动）。

#### 5.1 查看所有活动

显示系统中所有实践活动，包括：
- 活动名称
- 活动状态（招募中/进行中/已结束）
- 负责教师
- 报名人数
- 活动时间

#### 5.2 编辑活动

1. 点击活动的"编辑"按钮
2. 修改活动信息
3. 点击"保存"更新

#### 5.3 删除活动

1. 点击活动的"删除"按钮
2. 确认删除（注意：删除活动会同时删除相关的报名、小组、任务、报告和成绩数据）

### 6. 公告管理

#### 6.1 发布公告

1. 点击"公告管理" → "发布公告"
2. 填写公告信息：
   - 公告标题
   - 公告内容
3. 选择状态：
   - 草稿：暂不发布
   - 已发布：立即发布，教师和学生可见
4. 点击"提交"保存

#### 6.2 管理公告

- 查看所有公告列表
- 编辑公告内容
- 删除过期公告
- 修改公告状态（草稿 ⇄ 已发布）

### 7. 系统日志

#### 7.1 查看操作日志

系统自动记录所有用户的操作，包括：
- 操作用户
- 操作类型（登录、创建活动、审核报名等）
- 操作时间
- IP地址
- 请求参数

#### 7.2 筛选日志

可以按以下条件筛选日志：
- 时间范围
- 用户名
- 操作类型
- 关键字搜索

#### 7.3 清理日志

- 批量删除日志
- 清空所有日志

### 8. 数据导入导出

#### 8.1 批量导入学生

1. 点击"数据导入导出" → "导入学生"
2. 下载Excel模板文件
3. 按模板格式填写学生信息
4. 上传Excel文件
5. 系统自动导入并显示结果

#### 8.2 批量导入教师

操作流程与导入学生类似。

#### 8.3 数据导出

- 导出学生信息为Excel
- 导出教师信息为Excel
- 导出活动数据
- 导出成绩数据

## 教师使用指南

### 1. 登录系统

使用教师账号登录：
- 示例账号: `teacher1` / `123456`

### 2. 教师首页

登录后进入教师首页，显示以下功能：

```
├── 首页
├── 发布实践活动
├── 我的活动
├── 报名管理
├── 学生日常任务
├── 实践报告审核
├── 成绩评定
└── 系统公告
```

首页还会显示：
- 最新公告（管理员发布）
- 待处理报名数量
- 待审核报告数量
- 我负责的活动数量

### 3. 发布实践活动

#### 3.1 创建活动

1. 点击"发布实践活动"
2. 填写活动信息：
   - **活动名称**：活动的标题
   - **活动类型**：如"三下乡"、"企业实习"、"志愿服务"等
   - **活动描述**：详细说明活动内容、要求
   - **开始时间**：活动开始日期和时间
   - **结束时间**：活动结束日期和时间
   - **活动地点**：活动举办地点
   - **招募人数**：活动最大参与人数
   - **指导老师**：可选择多位教师共同负责（默认包含自己）
3. 点击"提交"发布活动

**注意**：
- 创建活动后，您会自动成为该活动的负责教师
- 可以选择其他教师作为共同负责人
- 活动初始状态为"招募中"

#### 3.2 活动状态说明

- **招募中**：学生可以报名
- **进行中**：活动正在进行，学生可以报名（如未满员）
- **已结束**：活动结束，学生不能报名，教师可以评定成绩

### 4. 我的活动

查看和管理自己创建或负责的活动。

#### 4.1 查看活动列表

显示所有我负责的活动，包括：
- 活动名称
- 状态
- 报名人数 / 招募人数
- 开始/结束时间

#### 4.2 编辑活动

1. 点击活动的"编辑"按钮
2. 修改活动信息
3. 可以修改活动状态：
   - 招募中 → 进行中
   - 进行中 → 已结束
4. 点击"保存"更新

#### 4.3 查看报名学生

1. 点击活动的"查看报名"按钮
2. 查看所有报名该活动的学生
3. 显示报名状态：
   - 待审核
   - 已通过
   - 已拒绝

#### 4.4 查看小组

1. 点击活动的"查看小组"按钮
2. 查看该活动下所有学生小组
3. 查看每个小组的成员信息

#### 4.5 删除活动

仅能删除自己创建的活动。

### 5. 报名管理

审核学生的活动报名申请。

#### 5.1 查看待审核报名

系统只显示以下活动的报名：
- 状态为"招募中"或"进行中"的活动
- 我负责的活动

**重要**：已结束的活动不再显示，不可操作。

#### 5.2 审核报名

对于每条待审核的报名，可以进行以下操作：

1. **通过**：
   - 点击"通过"按钮
   - 学生报名状态变为"已通过"
   - 学生可以创建或加入小组
   
2. **拒绝**：
   - 点击"拒绝"按钮
   - 学生报名状态变为"已拒绝"
   - 学生不能参与该活动

#### 5.3 批量审核

勾选多个待审核的报名，可以批量通过或拒绝。

### 6. 学生日常任务

查看学生提交的日常任务完成情况。

#### 6.1 按活动筛选

1. 选择具体活动
2. 查看该活动下所有学生提交的任务

#### 6.2 查看任务详情

每条任务显示：
- 学生姓名、学号
- 任务日期
- 任务内容
- 提交时间
- 状态（待处理/已完成）

### 7. 实践报告审核

审核学生提交的实践报告。

#### 7.1 查看待审核报告

显示状态为"待审核"的报告列表，包括：
- 学生姓名
- 活动名称
- 报告标题
- 提交时间
- 附件（如有）

#### 7.2 审核报告

1. 点击"审核"按钮
2. 阅读报告内容和附件
3. 填写反馈意见
4. 点击"提交审核"

审核后：
- 报告状态变为"已审核"
- 学生可以查看教师的反馈意见

#### 7.3 下载附件

如果学生上传了附件（Word、PDF等），点击附件链接可以下载查看。

### 8. 成绩评定

为已结束活动的学生评定成绩。

#### 8.1 选择活动

从下拉列表中选择一个"已结束"状态的活动。

#### 8.2 查看学生列表

显示该活动中所有"已通过审核"的学生，包括：
- 学生姓名、学号
- 已评分情况
- 平均成绩（如有多位教师评分）

#### 8.3 评分

对于每个学生：

1. 点击"评分"按钮
2. 填写评分表单：
   - **分数**：0-100分
   - **评语**：对学生表现的评价和建议
3. 点击"提交"保存

**多教师评分机制**：
- 如果活动由多位教师负责，每位教师都可以独立评分
- 系统自动计算所有教师评分的平均值
- 学生看到的是平均分和所有教师的评语

#### 8.4 修改评分

如果已经评过分，可以：
1. 点击"修改"按钮
2. 更新分数和评语
3. 点击"保存"更新

**注意**：只能修改自己的评分，不能修改其他教师的评分。

## 学生使用指南

#### 1.1 注册账号

1. 访问注册页面
2. 填写注册信息：
   - 用户名
   - 密码
   - 确认密码
   - 真实姓名
   - 角色：选择"学生"
   - 学号
   - 性别
   - 班级
   - 联系电话
   - 电子邮箱
3. 点击"注册"提交

#### 1.2 登录系统

使用注册的用户名和密码登录。

### 2. 学生首页

登录后进入学生首页，显示以下功能：

```
├── 首页
├── 浏览实践活动
├── 我的活动
├── 小组管理
├── 日常任务
├── 实践报告
├── 我的成绩
└── 系统公告
```

首页还会显示：
- 最新公告
- 已报名活动数
- 待完成任务数
- 已提交报告数

### 3. 浏览实践活动

查看所有可以报名的实践活动。

#### 3.1 活动筛选条件

系统只显示满足以下条件的活动：
- 状态为"招募中"或"进行中"
- 报名人数未达到上限

**注意**：已结束的活动不会显示在这里。

#### 3.2 查看活动详情

点击活动可以查看：
- 活动名称和描述
- 活动时间和地点
- 负责教师
- 招募人数 / 当前报名人数
- 活动要求

#### 3.3 报名活动

1. 点击"报名"按钮
2. 确认报名信息
3. 提交报名申请

报名后：
- 报名状态为"待审核"
- 等待教师审核
- 可以在"我的活动"中查看报名状态

### 4. 我的活动

查看已报名的活动和报名状态。

#### 4.1 查看报名状态

显示所有已报名的活动，包括：
- 活动名称
- 报名时间
- 报名状态：
  - ⏳ **待审核**：教师还未审核
  - ✅ **已通过**：可以参加活动，可以创建/加入小组
  - ❌ **已拒绝**：报名被拒绝

#### 4.2 改选活动

如果活动时间冲突，可以：
1. 取消当前报名
2. 重新报名其他活动

**限制**：
- 只能改选时间不冲突的活动
- 已通过审核的活动不建议随意取消

#### 4.3 退选活动

1. 点击"退选"按钮
2. 确认退选
3. 报名记录被删除

### 5. 小组管理

管理参加活动的实践小组。

#### 5.1 前置条件

**重要**：只有报名状态为"已通过"的活动才能创建或加入小组！

#### 5.2 创建小组

1. 点击"小组管理"
2. 从下拉列表选择一个"已通过"的活动
3. 点击"创建小组"
4. 填写小组信息：
   - 小组名称
5. 点击"提交"创建

创建小组后：
- 你自动成为组长
- 小组成员数为1
- 其他学生可以申请加入

**小组规则**：
- 每个小组最多10人
- 每个学生在同一活动中只能加入一个小组

#### 5.3 加入小组

1. 选择活动
2. 查看该活动下的所有小组
3. 选择要加入的小组
4. 点击"加入小组"

#### 5.4 退出小组

1. 在"我的小组"中找到小组
2. 点击"退出小组"
3. 确认退出

**注意**：如果你是组长，退出小组前需要先转让组长权限或解散小组。

#### 5.5 解散小组（仅组长）

1. 点击"解散小组"
2. 确认解散
3. 所有成员的小组关联被清除

#### 5.6 查看小组详情

点击"查看详情"可以看到：
- 小组名称
- 活动名称
- 组长信息
- 成员列表（姓名、学号、性别、班级、联系方式、邮箱）

### 6. 日常任务

提交和查看日常任务完成情况。

#### 6.1 提交任务

1. 点击"日常任务" → "提交任务"
2. 选择活动（从已通过的活动中选择）
3. 选择任务日期
4. 填写任务内容描述
5. 点击"提交"保存

#### 6.2 查看我的任务

显示所有已提交的任务，包括：
- 活动名称
- 任务日期
- 任务内容
- 状态（待处理/已完成）
- 提交时间

#### 6.3 编辑任务

对于"待处理"状态的任务，可以：
1. 点击"编辑"
2. 修改任务内容
3. 点击"保存"更新

### 7. 实践报告

提交实践报告并查看教师反馈。

#### 7.1 提交报告

1. 点击"实践报告" → "提交报告"
2. 选择活动（只能选择已参加的活动）
3. 填写报告信息：
   - 报告标题
   - 报告内容
   - 上传附件（Word、PDF等，可选）
4. 点击"提交"

**注意**：
- 每个活动只能提交一份报告
- 提交后状态为"待审核"

#### 7.2 查看报告状态

在报告列表中可以看到：
- 活动名称
- 报告标题
- 提交时间
- 审核状态：
  - ⏳ **待审核**：教师还未审核
  - ✅ **已审核**：教师已审核，可查看反馈

#### 7.3 查看教师反馈

如果报告已审核：
1. 点击"查看反馈"
2. 阅读教师的反馈意见
3. 根据反馈改进后续实践

### 8. 我的成绩

查看已结束活动的成绩和教师评语。

#### 8.1 查看成绩列表

显示所有已结束活动的成绩卡片，包括：
- 活动名称
- 活动时间和地点
- 平均成绩（多教师评分的平均值）
- 成绩等级（根据分数显示不同颜色）

#### 8.2 成绩详情

每个成绩卡片显示：

1. **平均成绩**：
   - 90分以上：绿色（优秀）
   - 75-89分：蓝色（良好）
   - 60-74分：橙色（及格）
   - 60分以下：红色（不及格）

2. **教师评分明细**：
   - 每位教师的评分
   - 每位教师的评语

3. **综合评语**：
   - 所有教师的评语汇总

## 常见操作流程

### 流程1: 学生参加活动的完整流程

```
1. 学生登录 → 浏览实践活动
   ↓
2. 选择感兴趣的活动 → 点击"报名"
   ↓
3. 报名状态变为"待审核" → 等待教师审核
   ↓
4. 教师在"报名管理"中审核 → 点击"通过"
   ↓
5. 学生报名状态变为"已通过"
   ↓
6. 学生进入"小组管理" → 选择该活动
   ↓
7. 创建新小组 或 加入已有小组
   ↓
8. 参与活动期间 → 提交日常任务
   ↓
9. 活动结束后 → 提交实践报告
   ↓
10. 教师审核报告 → 给出反馈
    ↓
11. 教师评定成绩 → 给出分数和评语
    ↓
12. 学生查看成绩和评语
```

### 流程2: 教师发布活动到评分的完整流程

```
1. 教师登录 → 发布实践活动
   ↓
2. 填写活动信息 → 选择共同负责的教师（可选）
   ↓
3. 提交发布 → 活动状态"招募中"
   ↓
4. 学生报名 → 教师审核报名
   ↓
5. 活动进行中 → 查看学生日常任务
   ↓
6. 活动结束 → 将活动状态改为"已结束"
   ↓
7. 审核学生实践报告 → 给出反馈意见
   ↓
8. 评定学生成绩 → 给出分数和评语
   ↓
9. （如有多位教师）其他教师也进行评分
   ↓
10. 系统自动计算平均分 → 学生查看成绩
```

### 流程3: 多教师协同评分流程

```
活动由3位教师共同负责：张老师、李老师、王老师

1. 活动结束后，张老师评分：
   - 学生A: 90分，评语: "表现优秀"
   ↓
2. 李老师评分：
   - 学生A: 88分，评语: "态度认真"
   ↓
3. 王老师评分：
   - 学生A: 92分，评语: "创新能力强"
   ↓
4. 系统自动计算：
   - 平均分 = (90 + 88 + 92) / 3 = 90.0分
   ↓
5. 学生查看成绩：
   - 平均分: 90.0分
   - 张老师: 90分 - 表现优秀
   - 李老师: 88分 - 态度认真
```

## ❓ 常见问题

### Q1: 为什么无法创建小组？

**A**: 请检查以下条件：
1. 是否已经报名了活动？
2. 报名状态是否为"已通过"？
3. 是否已经在该活动中加入了其他小组？

只有报名状态为"已通过"的学生才能创建或加入小组。

### Q2: 为什么看不到某个活动？

**学生端**：
- 只显示"招募中"或"进行中"的活动
- 只显示未满员的活动
- 已结束的活动不会显示在浏览列表中

**教师端**：
- 只显示自己负责的活动
- 可以在"我的活动"中看到所有状态的活动

### Q3: 报名被拒绝怎么办？

**A**: 如果报名被拒绝，可以：
1. 联系教师了解拒绝原因
2. 根据情况改进后重新报名
3. 或选择报名其他合适的活动

### Q4: 如何修改已提交的报告？

**A**: 如果报告还未被审核（状态为"待审核"），可以：
1. 先删除原报告
2. 重新提交新报告

如果报告已被审核，则无法修改，但可以与教师沟通。

### Q5: 成绩为什么是平均分？

**A**: 如果活动由多位教师共同负责，每位教师都会独立评分。系统会自动计算所有教师评分的平均值作为最终成绩。这样可以确保评分更加客观公正。

### Q6: 忘记密码怎么办？

**A**: 
- 联系管理员重置密码
- 或通过注册邮箱找回密码（如果系统开启了此功能）
